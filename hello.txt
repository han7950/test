04. Push 전에 Pull이 있다.

git pull origin master   --- 지어 해놓은 Repository에서 끌어옵니다.

git status  --- 현재 git 상태를 check

git add .변경된 파일들을 모두 Tracking

git commit -m "message" -- 인식 할 수 있는 commit을 달아줍니다.

git push origin +master  --  실제로 Git에 POUSH를 요청
